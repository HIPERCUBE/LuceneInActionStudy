# Searcher

## Start Indexer
```
Usage: java Searcher <index dir> <query>
```
You should input **index directory** and **query**.

```
<index dir> : /Indexer/result
<query>     : patent
```

## Query Example
 - patent
 - patent **AND** freedom
 - patent **AND NOT** apache
 - **+**copyright **+**developers
